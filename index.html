An always-on intelligence daemon hunting knife drops and Steel Flame releases across the web — so you never miss what matters.

Daemon Active simonhg321 / drop-watcher Est. 2026

01 — Mission

Score cool gear.
Be a good citizen.

Drop Watcher monitors maker websites, social feeds, and dealer pages for new releases and restocks in the Bushido mid-tech folder and Steel Flame jewelry space. It watches quietly, logs locally, and alerts only you — never hammering small makers, never bothering the community.

02 — Agents

Always On.
Never Sleeping.

AGENT / 01

Web Watcher

Polls maker and dealer websites on polite randomized intervals. Detects inventory changes, new listings, and restocks matching your cool list.

AGENT / 02

Feed Watcher

Consumes RSS feeds from Instagram, Facebook, and YouTube — watching for drop announcements and keyword matches from the makers you follow.

AGENT / 03

Orchestrator

Keeps everything alive. Manages rate limits, handles restarts, enforces the good citizen rules, and writes structured logs when something hits.

03 — Principles

The Code.

You Decide

The system watches and alerts. Every purchase decision belongs to the human. No automation crosses that line.

Polite by Design

Randomized intervals. One request at a time per domain. Robots.txt respected. Small makers protected.

Local First

Logs stay on this machine. No cloud sync, no third party data. Your intel is yours alone.

Config Driven

You own the cool list. Add makers, remove keywords, update sources — all in plain YAML. No code required.
